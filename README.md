## Описание проекта
Приложение для конвертации валют с графическим интерфейсом, использующее актуальные курсы из внешнего API. Сохраняет историю конвертаций и позволяет загружать её при следующем запуске.

## Функциональность
- ✅ Конвертация между различными валютами
- ✅ Получение актуальных курсов через API
- ✅ Сохранение истории конвертаций в JSON
- ✅ Загрузка истории при запуске
- ✅ Проверка корректности ввода (положительное число)
- ✅ Удобный интерфейс с таблицей истории

## Как получить API-ключ

1. Перейдите на сайт [ExchangeRate-API](https://www.exchangerate-api.com/)
2. Нажмите на кнопку "Get Free API Key"
3. Зарегистрируйтесь (укажите email и пароль)
4. После подтверждения email, вы получите бесплатный API-ключ
5. Бесплатный тариф позволяет 1500 запросов в месяц

**Важно:** Бесплатный ключ не требует ввода банковских данных!
